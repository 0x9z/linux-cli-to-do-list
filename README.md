# Terminal To-Do List

A simple Bash script to manage your daily tasks from the command line.

## Features
- ➕ Add new tasks with automatic ID and timestamp
- 📋 View all tasks stored
- 🗑️ Delete all tasks at once
- 🧹 Clean interface with auto-refresh
